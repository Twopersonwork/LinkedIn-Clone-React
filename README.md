# LinkedIn Clone

### Project is created with **Django REST Framework** and **React.js**
#### Features :
- Post (create, edit, delete, like, comment)
- Feed (posts of different users)
- Profile (edit,delete personal,educational info)
- Follow (follow, invite, ignore)
